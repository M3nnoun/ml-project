Welcome to the Student Math Score Prediction Project

This project uses machine learning to predict students' math scores based on various demographic and academic factors. By analyzing data from student performance datasets, we aim to provide insights into what influences math performance and offer a tool for predictions.

Whether you're an educator, student, or researcher, this tool can help understand patterns in student achievements and potentially guide interventions for better outcomes.

About the Dataset

The dataset used in this project includes information on student demographics and performance in exams. Key features include:

  • Gender: Sex of students (Male/Female)
  • Race/Ethnicity: Ethnicity of students (Group A, B, C, D, E)
  • Parental Level of Education: Parents' final education
  • Lunch: Standard or Free/Reduced
  • Test Preparation Course: Complete or Not complete
  • Math, Reading, Writing Scores
